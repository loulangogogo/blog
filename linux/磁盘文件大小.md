| 查看目标             | 核心命令   | 常用命令示例                             | 功能说明                                                 |
| :------------------- | :--------- | :--------------------------------------- | :------------------------------------------------------- |
| **系统硬盘（分区）** | `df`       | `df -h`                                  | 以易读格式（K,M,G）查看所有磁盘分区的使用情况。          |
|                      |            | `df -h /home`                            | 查看指定挂载点（如/home）的磁盘使用情况。                |
| **文件夹大小**       | `du`       | `du -sh /path/to/directory`              | 汇总（-s）显示指定目录的总大小，并以易读格式（-h）显示。 |
|                      |            | `du -h --max-depth=1 /path/to/directory` | 查看指定目录下**一级子目录和文件**的大小。               |
| **文件大小**         | `ls`/ `du` | `ls -lh filename.txt`                    | 以易读格式列出文件的详细信息，包括大小。                 |
|                      |            | `du -h filename.txt`                     | 直接显示文件的大小。                                     |
