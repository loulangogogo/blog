# 一、查看 ssh密钥是否存在

```sh
# 打开密钥目录
cd ~/.ssh
# 查看文件
ls
```

如果密钥存在就进入下一步，如果不存在就创建密钥：

>生成密钥
>
>```sh
>ssh-keygen -t rsa -b 4096 -C "注释说明用途"
>```
>
>- `-t rsa`：指定密钥类型为 RSA。
>- `-b 4096`：设置密钥长度为 4096 位（默认 3072 位，低于 2048 位的密钥已不安全）。
>- `-C "your_email@example.com"`：注释（通常用于标识密钥用途）。



# 二、将公钥传送到需要免密登录的服务器

```sh
ssh-copy-id -i id_rsa.pub root@xxx.xxx.xx.xx
```



> 目标服务器有了我的服务器公钥之后，后期就可以直接登录了。



# 三、连接服务器

```sh
SSH root@xxx.xxx.xx.xx
```
