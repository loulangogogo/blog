> sc query type= service | findstr /I MySQL

```cmd
C:\Users\Administrator>sc query type= service | findstr /I MySQL
SERVICE_NAME: MySQL80

```

> sc query MySQL80

## 三、忘记了用户名密码

### （一）、跳过权限的方式

#### 1. 进入C:\Program Files\MySQL\MySQL Server 8.0\bin目录下cmd

> mysqld --defaults-file="C:\ProgramData\MySQL\MySQL Server 8.0\my.ini" ^
> --skip-grant-tables ^
> --shared-memory ^
> --console

#### 2. 另外启动一个cmd窗口进入mysql

> mysql -u root

#### 3. 执行查询用户的命令查看用户

> SELECT user,host FROM mysql.user;

```cmd
mysql> SELECT user,host FROM mysql.user;
+------------------+-----------+
| user             | host      |
+------------------+-----------+
| admin            | %         |
| root             | %         |
| mysql.infoschema | localhost |
| mysql.session    | localhost |
| mysql.sys        | localhost |
+------------------+-----------+
```

#### 4. 修改/创建用户密码

修改

> FLUSH PRIVILEGES;
>
> ALTER USER 'root'@'%' IDENTIFIED BY '123456!';
>
> FLUSH PRIVILEGES;

创建

> FLUSH PRIVILEGES;
>
> CREATE USER 'admin'@'%' IDENTIFIED BY '123456';
> GRANT ALL PRIVILEGES ON *.* TO 'admin'@'%' WITH GRANT OPTION;
>
> FLUSH PRIVILEGES;

#### 5. 关闭mysqld的cmd退出



### （二）直接修改的方式

#### 1. 先创建reset.sql文件（里面是修改/创建用户的sql）

> ALTER USER 'root'@'localhost' IDENTIFIED BY '123456Aa!';

> CREATE USER 'admin'@'%' IDENTIFIED BY 'Admin@123';
> GRANT ALL PRIVILEGES ON *.* TO 'admin'@'%' WITH GRANT OPTION;

#### 2. 直接让mysql执行这个sql，启动cmd

> "C:\Program Files\MySQL\MySQL Server 8.0\bin\mysqld.exe" ^
> --defaults-file="C:\ProgramData\MySQL\MySQL Server 8.0\my.ini" ^
> --init-file="C:\reset.sql" ^
> --console

#### 3. 关闭退出这个cmd
