| 标准版本    | 头文件名          | 主要功能描述                                                 |
| :---------- | :---------------- | :----------------------------------------------------------- |
| **C89/C90** | `<assert.h>`      | 提供**断言**宏 `assert`，用于在调试时检查程序假设，若条件为假则输出错误信息并终止程序。 |
|             | `<ctype.h>`       | 提供用于**字符分类和转换**的函数，如判断字符类型（字母、数字等）或进行大小写转换。 |
|             | `<errno.h>`       | 提供**错误号**宏 `errno`及相关的错误代码宏，用于指示库函数执行过程中发生的错误。 |
|             | `<float.h>`       | 定义与**浮点类型**相关的各种极限值和特性的宏（如精度、最大最小值）。 |
|             | `<limits.h>`      | 定义与**整数类型**相关的各种极限值的宏（如 `INT_MAX`, `CHAR_BIT`）。 |
|             | `<locale.h>`      | 提供**本地化**相关函数和类型，用于处理与地域设置相关的信息，如数字、货币、时间的格式。 |
|             | `<math.h>`        | 提供大量**数学函数**，包括三角函数、指数、对数、幂运算、取整等。 |
|             | `<setjmp.h>`      | 提供**非局部跳转**功能，通过 `setjmp`和 `longjmp`实现跨函数跳转，通常用于异常处理。 |
|             | `<signal.h>`      | 提供**信号处理**功能，用于处理程序运行过程中产生的各种信号（如中断）。 |
|             | `<stdarg.h>`      | 提供处理**可变参数列表**的宏，用于定义可变参数函数（如 `printf`）。 |
|             | `<stddef.h>`      | 定义一些**标准定义**，如常用类型（`size_t`, `ptrdiff_t`）和宏（`NULL`, `offsetof`）。 |
|             | `<stdio.h>`       | 提供**标准输入输出**功能，是C语言中用于输入输出操作的核心库，支持文件操作和格式化I/O。 |
|             | `<stdlib.h>`      | 提供**通用工具函数**，包括内存管理、随机数生成、字符串转换、程序终止等。 |
|             | `<string.h>`      | 提供**字符串处理**函数，用于字符串的复制、连接、比较、搜索等操作。 |
|             | `<time.h>`        | 提供**日期和时间**处理函数，用于获取、转换和格式化时间。     |
| **C95**     | `<iso646.h>`      | 定义了一些替代关系操作符和位操作符的宏，如 `and`代替 `&&`，`or`代替 ` |
|             | `<wchar.h>`       | 定义**宽字符**支持，包括宽字符输入输出和字符串处理函数。     |
|             | `<wctype.h>`      | 提供**宽字符分类和映射**的函数，是 `<ctype.h>`的宽字符版本。 |
| **C99**     | `<complex.h>`     | 提供**复数**类型支持和相关的数学运算函数。                   |
|             | `<fenv.h>`        | 提供访问和操作**浮点环境**（如舍入模式、浮点异常标志）的函数。 |
|             | `<inttypes.h>`    | 提供对**精确宽度整数类型**（如 `int32_t`）的格式转换和输入输出支持，包含了 `<stdint.h>`的内容。 |
|             | `<stdbool.h>`     | 引入**布尔类型**支持，定义了 `_Bool`类型以及 `bool`, `true`, `false`宏。 |
|             | `<stdint.h>`      | 定义一组**精确宽度**的整数类型（如 `int8_t`, `uint32_t`），用于需要明确控制整数大小的场景。 |
|             | `<tgmath.h>`      | 提供**类型泛型数学宏**，根据参数类型自动选择 `<math.h>`或 `<complex.h>`中的合适函数。 |
| **C11**     | `<stdalign.h>`    | 提供与**内存对齐**相关的宏，如 `alignas`和 `alignof`的替代宏。 |
|             | `<stdatomic.h>`   | 提供**原子操作**的支持，定义了原子类型和原子操作函数，用于多线程环境中安全地访问共享数据。 |
|             | `<stdnoreturn.h>` | 定义 `noreturn`宏，用于标记那些**永不返回**的函数，有助于编译器优化和生成警告。 |
|             | `<threads.h>`     | 提供**多线程支持**，包括线程创建、互斥锁、条件变量等函数。   |
|             | `<uchar.h>`       | 提供 **Unicode 字符**支持（UTF-16 和 UTF-32 编码），定义了 `char16_t`和 `char32_t`类型。 |
| **C17**     | (无新增头文件)    | C17（ISO/IEC 9899:2018）**没有引入新的头文件**，主要是对C11标准进行技术修正和缺陷修复。 |
